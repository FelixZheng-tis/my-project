The whole search suggestions thing

So this is the complicated part, so there are ten search suggestions that are sitting invisible. when the user types something into the bar, all ten search suggestions automatically update, and if the bar is not empty, they will show.

now how do the search suggestions update?

well each item is actually stored as an object, with properties such as location and description. when search suggestions update, they basically filter through the list of all items in the Database and make a list of the top ten search results, then depending on which search suggestion they are (1 to 10) they will display that search result. Also they are buttons that onclick will update the info card (more on that on its own page)

Details

So essentially, "Database" is an array of all the items, we filter through the database using array.filter(boolean output function) and we filter to a array of only items whose name contains the text in the search bar using (.include).
