Lunch Brainstorming

So I was still eating lunch, and I started to think of ideas. We needed a sorting system, a system that could allow you to find things easy.

So I mentally laid out some objectives:
-It had to be easy to find
-Everything had to have a unique location that could be told
-the location has to be easily locatable

the idea

So obviously, in order to create a sorting system, it would be best to make a website that stores the information and allows you to search up stuff. the website would have to show a location, that means that every shelf and compartment needs a unique lable.

So basically, every shelf would be labled with a letter, and every compartment on that self would be numbered in order, that way finding the compartments would be easy too because greater numbers mean lower in the shelf.
And the website need to store every component/tool/thing in the lab along with their location.
Also it would be great to be able to show the components in a gallery when you're not searching, that way if you don't know what you are looking for you can just browse through stuff.

The pitch

But since this project involved the lab and reorganising everything, it was inmportant to get everyone, onboard. So I needed to make a pitch presentation. And so on monday, I presented to you and Mr. Fong.
