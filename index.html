Project Description

So my project is a website called Oogaboo, it is a website that is supposed to solve the problem of organisation in the robotics lab. It is a data storage, retreveal, and display system that is made to look very simple and easy to use. The project proposal describes that the main goals are that it is easy to use and user friendly, also it should be fast to use since it is meant to be a tool, and it should be somewhat easy to add new item to the system. Overall, I am pretty happy with how it turned out and I think I learned a lot, enjoy the website.
